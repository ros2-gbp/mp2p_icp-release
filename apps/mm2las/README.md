# mm2las

A command-line tool to export the layers of a MOLA metric map (*.mm) file as LAS point cloud files.

## Overview

`mm2las` is part of the MOLA project. It converts metric map files into the industry-standard LAS format (version 1.4), making it easy to visualize and process point cloud data in GIS applications, LIDAR processing tools, and 3D viewers like CloudCompare, QGIS, or ArcGIS.

Refer to the mm2las [documentation page](https://docs.mola-slam.org/latest/mp2p_icp_applications.html).
